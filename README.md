# WANDERLUST 🌍

WANDERLUST is a travel listing web application where users can explore, create, and review travel destinations.

## Features
- User authentication (login/signup)
- Add and manage listings
- Image upload support
- Reviews and ratings
- Responsive UI

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap

## Installation

Clone the repository:

git clone https://github.com/Subrat-2500/WANDERLUST.git

Install dependencies:

npm install

Run the server:

npm start
